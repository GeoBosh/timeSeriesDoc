# Installing timeSeries

The R package ‘timeSeries’ provides a time series class and tools for
creation, import, manipulation, statistical and financial computations
on time series.

Package `timeSeries` is part of the Rmetrics suite of R packages and is
developed on R-forge at
[timeSeries](https://r-forge.r-project.org/scm/viewvc.php/pkg/timeSeries/?root=rmetrics).
The root of Rmetrics is at
[R-forge](https://r-forge.r-project.org/projects/rmetrics).

Install the [latest stable
version](https://cran.r-project.org/package=timeSeries) of `timeSeries`
from CRAN:

``` R
install.packages("timeSeries")
```

You can install the [development
version](https://r-forge.r-project.org/scm/viewvc.php/pkg/timeSeries/?root=rmetrics)
of `timeSeries` from R-forge:

``` R
install.packages("timeSeries", repos = "http://R-Forge.R-project.org")
```

To report bugs visit
[Rmetrics](https://r-forge.r-project.org/projects/rmetrics/).

# Documentation

You can view the documentation of `timeSeries` at
[timeSeriesDoc](https://geobosh.github.io/timeSeriesDoc/) (an web site
produce with pkgdown) or download the [reference
manual](https://cran.r-project.org/package=timeSeries/timeSeries.pdf) of
the latest release from CRAN. There is also a large [vignette on
plots](https://cran.r-project.org/package=timeSeries/vignettes/timeSeriesPlot.pdf).
